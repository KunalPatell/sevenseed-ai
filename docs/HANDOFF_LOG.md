## 4. How to run / verify

```bash
# regenerate all marketing sites (from My Startups/)
python generate_sites.py

# compile-check any backend
cd "My Startups/apps/<slug>/backend" && python -m py_compile features.py main.py

# run an app locally (heavy imports = slow first start)
cd "My Startups/apps/<slug>/backend" && pip install -r requirements.txt && python main.py
# -> http://localhost:8000  (marketing site at /, APIs at /api/*, docs at /docs)
```

Optional env vars (in each `backend/.env`, gitignored): `GROQ_API_KEY`, `MISTRAL_API_KEY`
(LLM); `BREVO_API_KEY`+`BREVO_SENDER` (email); `WHATSAPP_ACCESS_TOKEN`+`WHATSAPP_PHONE_NUMBER_ID`
(WhatsApp); `ZOHO_CLIENT_ID/SECRET/REFRESH_TOKEN` (CRM, `.in` DC); `SERPAPI_KEY` (Emart prices);
`GNEWS_API_KEY`, `GITHUB_TOKEN`, `YOUTUBE_API_KEY`. All optional — everything degrades gracefully.
