FAQ

Building questions, answered

Which structural codes do you follow?

Every building design aligns with IS 456, IS 800, and National Building Code (NBC 2016). Defect reports include the relevant code clause.

How does image defect scanning work?

Upload a photo of any concrete structure. YOLOv8 (best.pt) classifies faults across 10 defect classes and drafts material requirements with costing in under 10 seconds.

Do I need an API key?

Yes — Argus uses BYOK. Add a free Gemini or OpenAI key and run unlimited reports. Without a key, the built-in offline estimator stays active.

Can I export estimates?

Yes. Download full structural timelines, BOQ checklists, and estimates as print-ready PDF reports from the project workspace.
